# BFC Portfolio

A modern interactive portfolio built with HTML, CSS, and JavaScript.

## Overview

This project is a personal development portfolio showcasing multiple interactive projects. Each project focuses on different programming concepts including input handling, timing systems, rendering, and persistent data storage.

## Features

- Animated particle background
- Responsive modern UI
- Interactive project navigation
- Clean and structured layout
- Built entirely with vanilla JavaScript

## Projects

- Click Speed Test — Measures user input speed
- Reaction Timer — Tests reflex timing accuracy
- Color Match — Logic-based interaction game
- Dodge Engine — Canvas-based movement and collision
- Smart Counter — Uses local storage for persistence

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Purpose

This portfolio was created to:
- Practice front-end development
- Experiment with interactive systems
- Showcase coding projects in a clean format
